# Healthcare Analytics Dashboard

## Project Overview
Developed an interactive Healthcare Analytics Dashboard using Power BI to analyze patient admissions, treatment outcomes, departmental performance, and operational KPIs.

## Tools Used
- Power BI
- DAX
- Power Query
- Excel

## Key Features
- Patient Admission Analysis
- Department-wise Performance Tracking
- KPI Monitoring
- Trend Analysis
- Interactive Filters and Slicers

## Key Insights
- Identified patient admission trends.
- Monitored hospital performance metrics.
- Improved visibility into healthcare KPIs.
